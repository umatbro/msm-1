Algorithm¶

Future state of an empty cell is determined by checking 4 rules:

  1. 5 or more cells from Moore neighbourhood
  2. 3 or more cells from nearest Moore neighbourhood
  3. 3 or more cells from further Moore neighbourhood
  4. any cell from neighbourhood if probability test is positive

Code¶

ca.neighbourhood.decide_by_4_rules(moore_neighbours: ca.neighbourhood.Neighbours, probability=50)[source]¶

Update grain field according to 4 consecutive rules:

  1. Moore (5-8 cells)
  2. Nearest Moore (min 3 neighbour cells)
  3. Further Moore (min 3 neighbour cells)
  4. Id of cell depends on cell of all neighbours and probability
Parameters:
  • moore_neighbours – all 8 Moore neighbours
  • probability – value used to calculate 4th rule
Returns:

output state of the cell (based on neighbours or None if state could not be chosen
